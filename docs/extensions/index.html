Extensions

Extensions provide to GNOME Shell additional features, or modify interface elements. When you enable an extension, you actually hot-loading code directly to GNOME Shell. Extensions can cause weird issues, and when that is the case you should unload the extension, and try again.

When you disable an extension from GNOME Tweak Tool the extension is trying to revert the default Shell code, but it is not unloaded. You should always restart GNOME Shell 

alt + f2 / r

In Wayland session, GNOME Shell cannot be restarted, so you need to relog.

Because there isnâ€™t an API for extensions, each extension can override another one, fully or partially, depending the loading order.

HOW DO I INSTALL AN EXTENSION?

First you need GNOME Tweak Tool 

>_ sudo apt-get install gnome-tweak-tool

You will find GNOME Shell extensions on EGO or in your Linux repositories. Starting from GNOME 3.22 extensions will be available via GNOME Software App.

Keep on mind that usually Github repos are far more updated than EGO or distro repos, therefore is good idea to directly get some extensions from there.

WHERE THE EXTENSIONS ARE INSTALLED?

On XDG-Folders. That will be 

/usr/share/gnome-shell/extensions      (system-wide)
~/.local/share/gnome-shell/extensions  (user-specific)

NOTICEABLE EXTENSIONS

The official GNOME Shell Extensions These are the extensions made by GNOME Project. 13 in total!

Dash to Dock A dock for the Gnome Shell. This extension moves the dash out of the overview transforming it in a dock for an easier launching of applications and a faster switching between windows and desktops.

Dynamic Transparency Adds Transparency To The Gnome Shell Panel, with a fade in / out effects for max / unmax windows

Workspaces to Dock A gnome shell extension that transforms the workspaces into an intellihide dock

Window Overlay Icons Add application icons to GNOME Shell window overview
